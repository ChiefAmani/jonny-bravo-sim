# Johnny Bravo: Joke Machine

## Overview
Get ready for some laughs with Johnny Bravo! This 2D interactive game brings the iconic character to life, delivering his signature jokes with accompanying animations. Just like the man himself, it's all about charm, confidence, and a whole lot of hair.

## Features
- **Johnny Bravo's Best Jokes**: A collection of classic Johnny Bravo-style one-liners.
- **Dynamic Animations**: Watch Johnny Bravo animate as he tells each joke, complete with his signature moves and expressions.
- **Interactive Fun**: Trigger new jokes with a simple click or key press.

## Setup Instructions
To get Johnny Bravo up and running on your machine, follow these simple steps:

1.  **Ensure Python is Installed**: This game requires Python 3.10.12 or a later stable 3.x version. You can download it from the official Python website: [python.org](https://www.python.org/downloads/)

2.  **Clone the Repository**: Open your terminal or command prompt and run the following command to download the game files:
    ```bash
    git clone https://github.com/ChiefAmani/jonny-bravo-sim.git
    cd jonny-bravo-sim
    ```

3.  **Install Dependencies**: The game uses the `arcade` library for 2D graphics and animation. Install it using pip:
    ```bash
    pip install -r requirements.txt
    ```
    (This will install `arcade==2.8.5` as specified in `requirements.txt`)

## How to Play
Once setup is complete, you're ready to enjoy Johnny Bravo's comedic stylings:

1.  **Run the Game**: Navigate to the game's directory in your terminal and execute the main script:
    ```bash
    python main.py
    ```

2.  **Trigger Jokes**: In the game window, simply **click your mouse or press any key** to make Johnny Bravo tell a new joke. Watch him animate and listen to his smooth delivery!
